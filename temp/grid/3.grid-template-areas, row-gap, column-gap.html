grid-template-areas, grid-area

  • grid-template-areas (container 속성)
    : 영역(Area) 이름을 참조해 템플릿 생성
    : "."을 이용해서 빈셀을 만들어 낼 수 있다.
  • grid-area (item 속성)
    : 영역(Area) 이름을 설정하거나, grid-row와 grid-column의 단축 속성

row-gap, column-gap

  • row-gap: (grid-row-gap) 행과 행 사이의 간격(Line)을 정의
  • column-gap: (grid-column-gap) 열과 열 사이의 간격(Line)을 정의
  • gap: (grid-gap) xxx-gap의 단축 속성

grid-template의 단축 속성 활용방법 2가지

1. grid-template는 2개로 나눠서 설정할 수 있는 단축 속성 입니다.

  • grid-template: [grid-template-rows] / [grid-template-columns];
  • grid-template: [grid-template-areas];
.container { grid-template: [grid-template-rows] / [grid-template-columns]; grid-template: [grid-template-areas]; }

2. grid-template는 다음 세개의 속성을 한번에 설정할 수 있는 단축 속성입니다.

  • grid-template는 다음 세개의 속성의 단축 속성입니다.
    (grid-template-rows, grid-template-columns, grid-template-areas)
  • code1, code2는 같은 결과물을 냅니다.
# code1 grid-template-rows: 80px 350px 130px; grid-template-columns: repeat(3, 100px); grid-template-areas: " header header header" ". main aside" "footer footer ."; # code2 grid-template: "header header header" 80px ". main aside" 350px "footer footer ." 130px / 100px 100px 100px;
